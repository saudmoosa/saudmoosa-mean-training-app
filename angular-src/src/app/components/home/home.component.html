Issue Tracking Application

A "M-E-A-N" stack application with JSON web token authentication deployed on Heroku

What goes into making this App

 
  •  

    Node.Js : Asynchronous event driven JavaScript runtime on the server side

     
  •  

    Express : Web application framework for Node.js

     
  •  

    MongoDB : Document-oriented cloud database fully managed by MLabs that hosts MongoDB

     
  •  

    Angular 6 : A TypeScript-based open-source front-end web application framework used with Material design

     
  •  

    Passport-JWT : Passport authentication middleware for Node.js with JSON web tokens strategy

     
  •  

    GitHub : A web-based hosting service for version control using Git where this code is publically available

     
  •  

    Heroku : A cloud platform as a service where this application is deployed with Continuous Integration from Git
